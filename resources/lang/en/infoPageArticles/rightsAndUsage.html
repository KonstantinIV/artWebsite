Rights And Usage

Usage: The received artwork can be used for personal use only unless otherwise agreed upon.

Personal use examples:

  • Printing single T-shirt, poster etc. for personal use.

  • Posting online, using it as a profile picture, etc.

Commercial use examples:

  • Selling prints, T-shirts etc.

  • Advertisement etc.

AI and NFT: Using the received artwork for NFT and AI-related projects is not allowed unless explicitly specified. If unsure of usages send email regarding your concerns.
